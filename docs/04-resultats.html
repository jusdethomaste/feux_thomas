4.1 Analyse de la structure spatiale

4.1.1 Fonction K de Ripley

La fonction \(K\) de Ripley permet de détecter des écarts à la complète aléatoire spatiale (CAS). Sous un processus de Poisson homogène, \(K(r) = \pi r^2\).

Code 
K_obs <- Kest(pp_homo, correction = "Ripley")

# Enveloppe Monte Carlo
env_K <- envelope(pp_homo, Kest, nsim = 99, correction = "Ripley", verbose = FALSE)

plot(env_K,
     main    = "Fonction K de Ripley",
     shadecol = "lightblue",
     lwd     = 2)

4.1.2 Fonction L

La transformation \(L(r) = \sqrt{K(r)/\pi}\) stabilise la variance et facilite l’interprétation. Sous la CAS, \(L(r) - r = 0\).

Code 
L_obs <- Lest(pp_homo, correction = "Ripley")

plot(L_obs, . - r ~ r,
     main = "Fonction L centrée",
     lwd  = 2,
     col  = "firebrick")
abline(h = 0, lty = 2, col = "grey40")

## Analyse de la structure spatiale

### Fonction K de Ripley

La fonction $K$ de Ripley permet de détecter des écarts à la complète aléatoire spatiale (CAS). Sous un processus de Poisson homogène, $K(r) = \pi r^2$.

```{r}
#| label: fig-ripley
#| fig-cap: "Fonction K de Ripley avec enveloppe de simulation (999 simulations)"

K_obs <- Kest(pp_homo, correction = "Ripley")

# Enveloppe Monte Carlo
env_K <- envelope(pp_homo, Kest, nsim = 99, correction = "Ripley", verbose = FALSE)

plot(env_K,
     main    = "Fonction K de Ripley",
     shadecol = "lightblue",
     lwd     = 2)
```

<!-- 
Commente ce que tu observes :
- Les points sont-ils dans l'enveloppe ? → cohérent avec la CAS
- Au-dessus ? → agrégation
- En dessous ? → régularité (inhibition)
-->

### Fonction L

La transformation $L(r) = \sqrt{K(r)/\pi}$ stabilise la variance et facilite l'interprétation. Sous la CAS, $L(r) - r = 0$.

```{r}
#| label: fig-Lest
#| fig-cap: "Fonction L centrée — écart à la complète aléatoire spatiale"

L_obs <- Lest(pp_homo, correction = "Ripley")

plot(L_obs, . - r ~ r,
     main = "Fonction L centrée",
     lwd  = 2,
     col  = "firebrick")
abline(h = 0, lty = 2, col = "grey40")
```
